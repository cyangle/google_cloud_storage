The Crystal module for the Cloud Storage JSON API

## Installation

### Install from Git

Add the following to shard.yaml

```yaml
dependencies:
  google_cloud_storage:
    github: cyangle/google_cloud_storage
    version: ~> 0.2.0
```

## Development

```shell
shards
```

```shell
crystal spec
```
